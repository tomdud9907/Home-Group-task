# Weather Dashboard

Minimal Next.js weather dashboard for the Home Group coding exercise.

## Setup

Install dependencies:

```bash
npm install
```

Create `.env.local` from `.env.example` and add your OpenWeather API key:

```bash
OPENWEATHER_API_KEY=your_api_key_here
```

Run the development server:

```bash
npm run dev
```

Open http://localhost:3000.
